# Product Board Mock Prompt

Use case: ui-mockup
Asset type: high-fidelity product UI mock for a Japanese B2B English vocabulary learning SaaS dashboard

Create a polished UI mockup for the Steady Study / MedAce web app redesign, showing both a desktop business administrator workspace and a mobile student learning dashboard in one clean presentation image.

Show the actual product surfaces: a desktop admin dashboard with next required action, activation progress, instructor load, at-risk student list, writing queue metrics; and an iPhone-sized student dashboard with today's one primary learning action, mission card, weakness focus, writing submission card, and bottom quick navigation.

Use a neutral product design board, realistic high-fidelity SaaS UI, crisp typography, Japanese interface copy, warm white base, restrained product accent, slate text, and secondary status colors. Keep it dense, calm, scannable, and operational rather than marketing-like.

Verbatim copy examples:

- 次の一手
- 要フォロー
- 今日やることは1つだけ
- 自由英作文
- 講師負荷
- 提出する

Avoid decorative blobs, stock photos, dark theme, oversized rounded cards, generic placeholder text, watermark, and unreadable microtext.
